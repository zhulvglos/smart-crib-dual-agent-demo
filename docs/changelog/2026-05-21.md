## 修复 OpenCV 中文乱码问题

- 问题：cv2.putText() 不支持 CJK 字符，所有中文显示为 "????"。
- 解决方案：创建 `utils_chinese_text.py` 工具模块，使用 PIL/Pillow + 系统字体（微软雅黑 msyh.ttc）渲染中文。
- 提供 `cv2_puttext_cn()` 函数，支持与 cv2.putText 类似的调用方式，额外支持 "stroke" 描边模式。
- 修改 demo_full_stage.py 的 4 处中文渲染：
  1. `_draw_text_shadow()` 静态方法 — 所有阶段提示文字（熟睡/苏醒/哭闹/危险④等）
  2. 顶部标题 "婴儿床监护 · 全阶段演示"
  3. 运动标签 "运动:"
  4. 区域图例（安全区/警告区/床边界）
  5. 实时监控 "实时监控中"
- 使用 ANCHOR="lt" 坐标系（与 cv2.putText 默认的 bottom-left 不同），调整了纵坐标。
- 自动探测系统字体路径（Windows: msyh.ttc, macOS: PingFang, Linux: WQY）。
